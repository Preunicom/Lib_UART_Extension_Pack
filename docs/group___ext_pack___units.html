Detailed Description

Defines unique identifiers for ExtPack units.

Each unit is assigned a unique numerical identifier from 0 to 63. These constants are used throughout the UART Extension Pack to refer to specific hardware units.

Example usage:

unit_t my_unit = unit_U05;

typedef uint8_t unit_t

Type alias for unit identifier.

'unit_t' is defined as uint8_t and is used to represent the unique identifiers for various units within the system. It is passed to functions that require a specific unit identifier for communication or management.
