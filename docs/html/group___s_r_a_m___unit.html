uint8_t * recv_data,

Reads the data from the SRAM unit of the previously set address. Waits for the result for a maximum time set by timeout. The recv_data value is only valid if the operation terminated successfully.

Parameters
unitThe SRAM unit to read the data from.
recv_dataPointer to store the received data to.
timeout_usThe maximum time to wait for the data in us.
Returns
EXT_PACK_SUCCESS on success, EXT_PACK_FAILURE on failure.
