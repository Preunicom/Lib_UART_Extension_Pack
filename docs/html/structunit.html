void(* unit::custom_ISR) (unit_t, uint8_t)

Pointer to a custom interrupt service routine (ISR).

This function is called by the system when the unit triggers an interrupt as callback function. The parameters are the unit identifier and the associated data byte.

Warning
Interrupts will not be enabled before calling the custom_ISR.
