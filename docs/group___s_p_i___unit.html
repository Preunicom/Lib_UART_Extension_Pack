◆ send_ExtPack_SPI_String_to_slave()

ext_pack_error_t send_ExtPack_SPI_String_to_slave ( unit_t unit,
uint8_t slave_id,
const uint8_t * data,
uint16_t send_byte_delay_us )

Sends a set_ExtPack_SPI_slave control message followed by the given String until '\0' to ExtPack with send chosen mode. If a send char operation fails the function aborts and returns an error.

Parameters
unitThe ExtPack unit to which the data should be sent.
slave_idThe slave id to send the data to.
dataThe data to be sent as String with terminating '\0'.
send_byte_delay_usThe delay between sending two bytes in us.
Returns
EXT_PACK_SUCCESS on success, EXT_PACK_FAILURE if the function was aborted when sending a char because of an error while sending.
