Detailed Description

Advanced extension functions for the ExtPack library.

This header provides higher-level helper functions for ExtPack units, including utilities to send strings over ExtPack with defined byte delays.

Provided Functions:

  • send_String_to_ExtPack: Send null-terminated strings with a specified delay between bytes.

ext_pack_error_t send_String_to_ExtPack ( unit_t unit,
const uint8_t * data,
uint16_t send_byte_delay_us )

Sends the given String until '\0' to ExtPack. If a send char operation fails the function aborts and returns an error.

Parameters
unitThe ExtPack unit to which the data should be sent. Including the correct set access mode for sending.
dataThe data to be sent as String with terminating '\0'.
send_byte_delay_usThe delay between sending two bytes in us.
Returns
EXT_PACK_SUCCESS on success, EXT_PACK_FAILURE if the function was aborted when sending an uint8_t because of an error while sending.
