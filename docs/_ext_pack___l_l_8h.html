Detailed Description

Low-Level hardware interaction layer for ExtPack communication.

This file provides functions for direct communication with ExtPack units via UART, including unvalidated command transmission and interrupt handling.

Features:

  • Low-level initialization of hardware resources.
  • Raw UART command transmission to ExtPack units.
  • Basic critical section handling using interrupt control.

This layer operates without validation or abstraction and is used internally by higher-level ExtPack logic.

◆ exit_critical_zone()

void exit_critical_zone ( )

Resets the interrupt state to the last saved one from enter_critical_zone.

This function resets the status register to the saved value. The value was saved when calling enter_critical_zone(). This is used to exit a critical zone.

Warning
Do not use before calling enter_critical_zone()!
